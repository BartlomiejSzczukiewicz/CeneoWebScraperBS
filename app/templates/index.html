About This Project

The Ceneo.pl Web Scraper is designed to extract and list product opinions from the Ceneo.pl website. The application utilizes various Python libraries to perform web scraping and data analysis.

Purpose of the Application

This application aims to provide users with a convenient way to gather and analyze product opinions from Ceneo.pl, helping them make informed purchasing decisions.

Application's Operation

The application works by scraping product opinions from the Ceneo.pl website and presenting them in a user-friendly format. Users can extract opinions for specific products and view a list of all products for which opinions have been extracted.

Libraries Used

  • BeautifulSoup
  • Requests
  • Pandas
  • Flask

Applied Approaches

The application employs web scraping techniques, data cleaning, and visualization to present the extracted opinions effectively.
